Second-hand website

A campus-wide secondand goods trading website targeting students and teachers.

  • Learn more

Project background

Each semester,students have many unused items(like textbooks,furnture,appliances,etc),especially for graduates. Some students conduct transactions through WhatsApp groups, online forums, Telegram, etc. These methods are not transparent and not secure. So we need to create a secure platform.

Project goals

For students, some new items are too expensive and won't be used for a long time. Therefore, a second-hand item is very suitable for some students.

Project advantage

1.Information disclosure

2.Payment security

3.Pay on delivery

4.Seven-day no-questions-asked refund

Platform framework

The platform stores user, product, and transaction data in a database and utilizes features such as identity verification, item display, and security.

Student identity verification

This platform is only available for students and teachers on campus to use.By verifying the students' ID and the teacher's number, it is determined whether they are from this school.

Product Listing and Browsing

You can browse the products after passing the identity verification.And The platform stores basic product information and displays it when users browse. Users can filter products by category, price, or keywords.

Instant Messaging

Provides instant chat so that buyers and sellers can confirm item details and handover plans.

Transaction Management

After the buyers and sellers agree on the price, they can then proceed with the transaction.The system records both users, transaction status, and timestamps to keep the process clear.Supports face-to-face handover on campus and saves brief handover details for safety.

Content Safety and Review

Reports and review results are recorded to help the backend handle suspicious users or illegal products.

Transaction Safety and Credit

Funds in escrow are kept safe until the transaction is confirmed.Reviews after each transaction are saved to build usersâ€™ credit scores.
